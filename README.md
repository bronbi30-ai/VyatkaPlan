markdown # ВяткаПлан (VyatkaPlan) Цифровая система развития компании ВяткаПлан. Проект предназначен для объединения инженерных разработок, работы с клиентами, маркетинга и рабочей документации в единой системе. --- # Основное направление проекта ## VPI-6K Разработка мобильной инсинераторной печи ВПИ-6К. Раздел включает в себя: - разделение изделий; - свет; - 3D-модели; - чертежи; - расчеты; -испытания; - подготовка производства. --- ## CRM Система управления клиентами и заказами. В разделе хранения: - информация о клиентах; - заявки; - история взаимодействия; - коммерческие предложения; - статусы заказов. --- ## ВК_Контент Материалы для продвижения компании. Раздел включает в себя: - идеи публикации; - готовые посты; - фотографии; - кейсы выполненных работ; - рекламные материалы. --- ## Документы Документация Рабочая компания: - инструкция; - шаблоны; - договоры; - технические документы. --- ## Фото_Видео Медиаматериалы: - фотографии техники; - фото выполненных работ; - видеопроцессы; - материалы для публикации. --- # Принципы работы проекта 1. Все важные изменения фиксируются через Git. 2. Документы имеют понятную структуру и историю изменений. 3. Новые идеи сначала фиксируются, а затем реализуются. 4. Клод используется в качестве помощника для анализа, проектирования, документации и развития компании. --- # Текущий статус Проект находится на стадии формирования структуры и направления разработки ВПИ-6К. Дата создания: 2026 год
